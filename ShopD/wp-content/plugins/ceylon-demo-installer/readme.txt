== Description ==

Setup your site theme from [Ceylon Themes](https://www.ceylonthemes.com/) site with template library dummy data easily. Import settings, widgets and content with one click. Ceylon Demo Setup requires [Advanced Import](https://wordpress.org/plugins/advanced-import/) Plugin to work normally.

While you use Ceylon Demo Setup to import demo starter site, Images and demo files are fetches from respected theme Demo Sites form Ceylon Themes. This helps you to import starter dmeo site with a single click. You must accept [terms](https://www.ceylonthemes.com/terms-and-conditions/) and [privacy](https://www.ceylonthemes.com/privacy-policy/) to use Ceylon Demo Setup Plugin.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'ceylon-demo-installer'
3. Activate Ceylon Demo Setup from Appearance > Plugins
4. Go to the Appearance -> Demo Import
5. Use available demo import

= From WordPress.org =

1. Download ceylon-demo-installer.zip
2. Upload the 'ceylon-demo-installer' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate ceylon-demo-installer from your Appearance > Plugins page.
4. Go to the Appearance -> Demo Import
5. Use available demo import

== Frequently Asked Questions ==

= I have activated the plugin. Where to find the plugin page to import data? =

You will find the import page in *wp-admin -> Appearance -> Demo Import*.
